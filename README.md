# Brokart

A full-featured ecommerce web application built with Django.

## Features

- Product listing
- Cart system
- Order management
- Payment

## Tech Stack

- Python
- Django
- SQLite
- PosgreSQL

## Setup Instructions

```bash
git clone <https://github.com/nithin-k-m/brokart.git>
cd brokart_project
python -m venv projectenv
projectenv\Scripts\activate
pip install -r requirements.txt
python manage.py runserver


## Run Locally
Open your browser and go to:
http://127.0.0.1:8000/
